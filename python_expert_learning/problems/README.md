## ğŸ“ Problem-Solving Patterns

### 1. Two Pointers
- Use when: Array is sorted, need to find pairs
- Template: Left and right pointers moving towards center

### 2. Sliding Window
- Use when: Subarray/substring problems
- Template: Expand window, shrink when condition met

### 3. Hash Map
- Use when: Need O(1) lookups
- Template: Store seen values, check for complements

### 4. DFS/BFS
- Use when: Tree/Graph traversal
- Template: Recursive DFS or Queue-based BFS

### 5. Dynamic Programming
- Use when: Overlapping subproblems
- Template: Memoization or tabulation
